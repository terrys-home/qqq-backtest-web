# Step24A 라오어 무한매수4.0 원문기반 새 엔진

이번 버전은 기존 V4/V5 근사 엔진을 고치는 것이 아니라, 라오어 무한매수법 4.0 방법론 문장 기준으로 새 엔진을 추가한 버전입니다.

## 반영한 원문 규칙

- 시작 전 보유수량 0이면 `T=0`
- 첫매수는 1회매수금으로 전날종가보다 적당히 큰 값으로 LOC 매수 시도
- 첫매수 LOC 여유율은 원문 가이드 `전날종가 대비 +10~15%`, 기본값은 `12%`
- 일일매수시도액 = `잔금 / (분할수 - T)`
- TQQQ/SOXL 20/40분할 별% 공식 반영
- 별%가격 = `평단 × (1 + 별%/100)`
- 매수점 = `별%가격 - 0.01`
- 매도점 = `별%가격`
- 전반전 매수 = 별지점/큰수 LOC 0.5T + 평단 LOC 0.5T
- 하나만 체결되면 `T + 0.5`, 둘 다 체결되면 `T + 1.0`
- 쿼터매도 = 보유수량 25% 매도 후 `T = 기존 T × 0.75`
- 지정가매도 = TQQQ +15%, SOXL +20%

## 의도적으로 임의 구현하지 않은 부분

- 30분할 별% 공식은 제공된 원문 이미지에 명시되지 않아 지원하지 않았습니다.
- TQQQ/SOXL 외 티커는 원문 공식이 명시되지 않아 지원하지 않았습니다.
- 후반전/소진모드 세부 LOC 수량표는 추가 원문 확인 후 더 정교화하는 구조로 두었습니다.
