# Step24G - 라오어 무한매수4.0 원문엔진 / 지정가 커스텀 입력

## 변경 요약

- Step24F 기반 유지
- 매일 매도예약 구조 유지
  - 보유수량 25%: 별지점 LOC 매도
  - 보유수량 75%: 평단 대비 지정가 매도
- 전량매도 지정가 수익률을 드롭다운이 아니라 숫자 입력칸으로 직접 입력 가능
- 기본값은 원문 기준 유지
  - TQQQ: 15%
  - SOXL: 20%
- 사용자가 10, 13.5, 17.2, 20 등 원하는 값을 직접 입력 가능

## 주의

- 지정가 수익률 커스텀 입력은 백테스트 비교용입니다.
- 원문 기본값을 기준으로 검증하려면 TQQQ 15%, SOXL 20%를 사용하세요.
- 별LOC 매도점이 지정가보다 높게 계산되는 경우 RaorDebug에 경고가 기록됩니다.
