# Step24Q RAOR Original Engine

Step24Q는 Step24P의 원문 엔진 구조를 유지하면서 실전 편의 기능을 강화한 버전입니다.

## 원칙
- 라오어 무한매수4.0 원문 기준 엔진은 변경하지 않습니다.
- 원문에 없는 매매 공식은 추가하지 않습니다.
- Step24Q의 변경 범위는 화면/캐시/CSV/후보 적용/진행형 UI 고도화입니다.

## 추가 기능
- Step24Q 전략 선택 옵션 추가
- Optimizer TOP100 / DeepMining TOP50 캐시를 Step24 전용 폴더에 분리 저장
- 같은 조건이면 재계산하지 않고 캐시 재사용
- 캐시 CSV 다운로드 링크 추가
- TOP1/TOP2/TOP3 빠른 적용 버튼 추가
- 결과표의 적용 링크 클릭 시 설정값 자동 반영 및 백테스트 실행
- 현재 적용 후보 표시
- 진행형 오버레이에 예상 조합 수와 실행 모드 표시
- 결과표 1위/판정별 가독성 강화
