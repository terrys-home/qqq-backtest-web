# Step24F - 라오어 무한매수4.0 원문엔진 매도예약 검증판

## 핵심 수정

- Step24E의 매일 매도예약 구조를 유지합니다.
- 보유수량 1/4은 별지점 LOC 매도, 나머지 3/4은 지정가 매도로 계산합니다.
- 전량매도 지정가 기준을 UI 드롭다운으로 선택할 수 있게 했습니다.
  - 원문 기본값: TQQQ 15%, SOXL 20%
  - 비교값: 10%, 12%, 15%, 20%, 25%, 30%
- 별LOC 매도점이 지정가 매도점보다 높게 계산되면 `RaorDebug`에 경고를 남깁니다.

## 매도 체결 근사

- 지정가 매도: 당일 고가가 지정가 이상이면 체결
- 별지점 LOC 매도: 당일 종가가 별지점 이상이면 체결
- 같은 날 둘 다 충족은 다음 의미입니다.
  - 고가가 지정가 이상
  - 종가가 별지점 이상
  - 별지점 가격이 지정가보다 높다는 의미가 아닙니다.

## 주의

- 후반전 세부 사다리 수량표는 아직 임의로 넣지 않았습니다.
- 30분할 별% 공식은 원문 확인 전까지 지원하지 않습니다.
- TQQQ/SOXL 20/40분할 공식만 사용합니다.
