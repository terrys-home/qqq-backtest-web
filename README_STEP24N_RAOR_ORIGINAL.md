# Step24N RAOR Original Engine

Step24M 기반 정리판입니다.

## 변경사항
- Step24A~Step24D 실험 파일 제거
- 전략 드롭다운에서 Step24A~D 제거
- 현재 사용 전략은 Step24N/M/L/H/G/F/E 중심으로 정리
- Step24N은 Step24M 로직을 유지한 정리판입니다.

## 유지 로직
- 별LOC 매수 = 별지점 - 0.01
- 별LOC 매도 = 별지점
- 보유수량 25% 별LOC 매도
- 보유수량 75% 평단 대비 지정가 매도
- 지정가 % 직접 입력
- 가격겹침 오류 / 일봉 선후관계 불명 / 완료누락 카운터 유지
