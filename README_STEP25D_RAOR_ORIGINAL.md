# Step25D - 무한매수법 2.2 통합전략 / 4.0 원칙 재검증판

## 기준
- 기존 zip 해석보다 사용자가 새로 제공한 사진/정리 순서를 우선합니다.
- 메인 백테스트 대상은 무한매수법 V2.2와 V4.0입니다.
- V3.0은 참고용이며 구현 메인이 아닙니다.
- 메인 티커는 TQQQ / SOXL입니다.

## 현재 Step25C/V4 계열 엔진 점검 요약
- V4.0 일반모드 중심으로 구현되어 있습니다.
- 별% 공식은 TQQQ/SOXL, 20/40분할 기준으로 분리되어 있습니다.
- 매수 LOC = 별가격 - 0.01, 매도 LOC = 별가격 구조는 반영되어 있습니다.
- 전반전: 별LOC 0.5T + 평단LOC 0.5T 구조가 반영되어 있습니다.
- 후반전: 별LOC 1T 구조가 반영되어 있습니다.
- 매도: 별LOC 25% + 지정가 75% 구조가 반영되어 있습니다.
- 리버스모드는 아직 정식 구현이 아니며 별도 단계가 필요합니다.

## Step25D에서 보강한 점
- V2.2를 V4.0 코드에 섞지 않고 `muhan_raor_v22_engine.py`로 분리했습니다. 단, V2.2 기본형/수치변화는 나누지 않고 하나의 통합 전략으로 취급합니다.
- 드롭다운에 `라오어 무한매수법 2.2 통합 원문엔진`을 추가했습니다.
- 기본 전략을 `라오어 무한매수4.0 원문엔진 / Step25D 검증판`으로 변경했습니다.
- V2.2 엔진은 하나의 전략 안에서 TQQQ/SOXL 공식과 지정가 10%/12% 차이를 자동 반영합니다.

## V2.2 통합 엔진 반영 원칙
- TQQQ 별% = 10 - T/2 × (40/a)
- SOXL 별% = 12 - T×0.6 × (40/a)
- 별가격 = 평단 × (1 + 별%/100)
- 매수 LOC = 별가격 - 0.01
- 매도 LOC = 별가격
- 전반전: 0% LOC 0.5T + 별LOC 0.5T
- 후반전: 별LOC 1T
- 매도: 별LOC 25% + 지정가 75%
- TQQQ 지정가 10%, SOXL 지정가 12%
- 큰수매수는 주문거부/RPA 예외이므로 기본 백테스트에는 넣지 않았습니다.
