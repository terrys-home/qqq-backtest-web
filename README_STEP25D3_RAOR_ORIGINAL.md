# Step25D-3 RAOR Original Engine

## 목적
전략 선택 화면을 실전 검증용으로 정리한 버전입니다.

## 변경 사항
- 기본 티커를 SOXL로 변경했습니다.
- 티커 드롭다운은 SOXL, TQQQ만 표시합니다.
- 기본 전략을 무한매수법 2.2 통합 원문엔진으로 변경했습니다.
- 전략 선택 드롭다운에서 이전 연구용 전략과 Step24/Step25 역사 버전들을 숨겼습니다.
- 전략 선택에는 아래 2개만 남겼습니다.
  1. 라오어 무한매수법 2.2 통합 원문엔진
  2. 라오어 무한매수법 4.0 원문엔진 / Step25D 검증판
- 기본 분할횟수를 40으로 변경했습니다.

## 유지 원칙
- 2.2와 4.0만 메인 백테스트 대상입니다.
- V3.0은 참고용이며 전략 선택에 노출하지 않습니다.
- SOXL/TQQQ만 메인 티커로 사용합니다.
- 기존 연구용 코드 파일은 백업 목적으로 보존하지만, UI에서는 숨깁니다.
