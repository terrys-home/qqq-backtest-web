# Step25D-2 - 무한매수법 2.2 통합전략 / 4.0 원칙 재검증판

## 기준 변경
- 사용자가 새로 정리한 기준에 따라 **무한매수법 2.2는 하나의 통합 전략**으로 취급합니다.
- `2.2 기본형`과 `2.2 수치변화`를 별도 전략/버전으로 나누지 않습니다.
- 티커가 TQQQ인지 SOXL인지에 따라 내부 공식만 자동으로 달라집니다.
- 기존 zip 해석보다 사용자가 새로 제공한 사진/정리 순서를 우선합니다.
- 메인 백테스트 대상은 무한매수법 2.2와 4.0입니다.
- V3.0은 참고용이며 구현 메인이 아닙니다.
- 메인 티커는 TQQQ / SOXL입니다.

## Step25D-2에서 정리한 점
- `muhan_raor_v22_engine.py`의 설명과 UI 문구를 V2.2 통합전략 기준으로 정리했습니다.
- 드롭다운 표기를 `라오어 무한매수법 2.2 통합 원문엔진`으로 변경했습니다.
- SOXL을 별도 “수치변화 버전”으로 표시하지 않고, V2.2 안의 티커별 공식으로 처리합니다.
- V4.0 엔진과 V2.2 엔진은 계속 분리합니다.
- 큰수매수는 주문거부/RPA 예외이므로 기본 백테스트에는 넣지 않습니다.

## V2.2 통합 엔진 공식
### TQQQ
- 별% = `10 - T/2 × (40/a)`
- 지정가 매도 = `10%`
- 쿼터손절 LOC = `-10%`

### SOXL
- 별% = `12 - T×0.6 × (40/a)`
- 지정가 매도 = `12%`
- 쿼터손절 LOC = `-12%`

## V2.2 공통 원칙
- 별가격 = 평단 × (1 + 별%/100)
- 매수 LOC = 별가격 - 0.01
- 매도 LOC = 별가격
- 전반전: 0% LOC 0.5T + 별LOC 0.5T
- 후반전: 별LOC 1T
- 매도: 별LOC 25% + 지정가 75%
