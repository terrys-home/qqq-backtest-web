# Step25E-3 - V4.0 T / 1회매수금 / 체결 이벤트 검증판

목적:
- V4.0 일반모드 수익률 평가 전에 T 변화, 1회매수금, 체결 이벤트 기록이 원문 기준으로 검증 가능한지 확인한다.
- 새 매매공식은 추가하지 않는다.
- V2.2 검증마감판과 V4.0 일반모드를 분리한다.
- 리버스모드는 아직 제외한다.

추가 화면:
- Step25E-3 V4.0 T / 1회매수금 / 체결 이벤트 검증
- T계산 실패
- T범위 실패
- 1회매수금 실패
- 첫/전반/후반 매수 카운트
- 지정가/별LOC 매도 카운트
- 동일일 매수·매도 카운트
- OHLC 선후불명 카운트
- 최근 체결/주문 이벤트 샘플

검증 기준:
- TAfter - TBefore - TDelta 일관성
- T가 0~분할수 범위를 벗어나지 않는지
- DailyAttemptBefore/After가 NaN 또는 음수가 아닌지
- 첫매수/전반전/후반전/지정가매도/별LOC매도 이벤트가 로그에서 확인되는지
- 같은 날 매수·매도 동시 실행 여부
- BuyLOC >= SellLOC 가격 겹침 여부

주의:
- 이 단계는 결과가 좋은지 판단하는 단계가 아니라 V4.0 엔진 로그 검증 단계다.
- 원문에 없는 공식/수량/체결 규칙은 추가하지 않는다.
