# Step25E-3-2 - V4.0 지정가 원문값 고정 + T/체결 검증판

## 수정 이유
Step25E/E-2/E-3에서는 V4.0 화면에 `지정가 매도(원문 고정) 20%`처럼 표시되더라도,
URL 또는 이전 프리셋에 `designated_sell_pct=10` 같은 값이 남아 있으면 엔진 실행에는 그 값이 전달될 수 있었다.
따라서 같은 V4.0 화면처럼 보여도 결과가 달라지는 문제가 있었다.

## 수정 내용
- `strategy == raor4_step25e`일 때 엔진 실행 전 지정가 매도율을 원문값으로 강제 고정한다.
  - TQQQ: 15%
  - SOXL: 20%
- URL/프리셋 잔여 `designated_sell_pct` 값은 V4.0 Step25E 재검증판에서 무시한다.
- T/1회매수금/체결 이벤트 검증표는 유지한다.
- 매매 공식 자체는 추가하지 않았다.

## 확인 포인트
- SOXL 40분할 V4.0 일반모드에서 지정가 매도율이 20%로 고정되는지
- URL에 `designated_sell_pct=10`이 남아 있어도 결과가 흔들리지 않는지
- E-3 검증표의 T계산 실패, T범위 실패, 1회매수금 실패를 계속 확인한다.
