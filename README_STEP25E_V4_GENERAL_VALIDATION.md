# Step25E - V4.0 일반모드 원문 기준 재검증판

## 목적

Step25D-8에서 V2.2 통합 원문엔진의 로그, 쿼터손절, 주기상태 검증을 마감했다.
Step25E는 V4.0 일반모드를 V2.2와 분리해서 다시 검증하기 위한 진입판이다.

## 기준

- V4.0은 V2.2와 별도 전략으로 검증한다.
- 리버스모드는 아직 제외한다.
- 기존 Step24X/O 계열 일반모드 엔진 로직을 유지한다.
- 새 공식, 새 수량표, 임의 체결 규칙은 추가하지 않는다.

## V4.0 일반모드 체크 항목

- TQQQ 20분할: 별% = 15 - 1.5T
- TQQQ 40분할: 별% = 15 - 0.75T
- SOXL 20분할: 별% = 20 - 2T
- SOXL 40분할: 별% = 20 - T
- 첫매수 LOC: 전일종가보다 10~15% 위
- 1회매수금: 첫 매수는 원금/분할수, 이후 잔금/(분할수-T)
- 전반전: 0.5T 별LOC + 0.5T 평단LOC
- 후반전: 1.0T 별LOC
- 매도: 1/4 별LOC + 3/4 지정가
- 지정가: TQQQ 15%, SOXL 20%
