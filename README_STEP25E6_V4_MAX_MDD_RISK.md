# Step25E-6 RAOR V4.0 Max MDD / OPEN Risk Audit

목적: V4.0 일반모드의 매매공식은 변경하지 않고, 결과 시계열에서 최대 MDD가 언제/어떤 주기에서 발생했는지 확인한다.

추가 항목:
- 전체 최대 MDD 발생일
- MDD 발생 주기 / 당시 T / 당시 보유평가액 / 현금 / 총자산
- 직전 고점자산과 회복일 / 회복 소요일
- 주기별 내부 최악 DD 상위 10개
- 현재 OPEN 주기의 최악 DD, 최대T, 진행일수

주의:
- 전체 MDD는 전체 TotalAsset 시계열 기준이다.
- 주기별 최악 DD는 각 주기 내부 고점 대비 하락률이므로 전체 MDD와 기준이 다르다.
- 공식/체결조건/수수료 계산은 Step25E-5와 동일하게 유지한다.
