# Step24E - 라오어 무한매수4.0 원문엔진 매도예약 보강

## 핵심 수정

Step24D까지는 완료 주기 보정 과정에서 별지점 LOC 매도 중심으로 주기 완료를 처리했습니다.
Step24E에서는 사용자가 제공한 원문 매도방법 기준을 반영했습니다.

## 반영한 원문 매도 구조

- 매도는 전반전/후반전 상관없이 공통 적용
- 매일 보유수량의 1/4은 별지점에 LOC 매도 예약
- 나머지 보유수량 3/4은 지정가 매도 예약
- 지정가 기준
  - TQQQ: 평단 대비 +15%
  - SOXL: 평단 대비 +20%
- 주간장은 사용하지 않는 전제

## 백테스트 체결 근사

- LOC 매도: 당일 종가가 별지점 이상이면 종가 체결
- 지정가 매도: 당일 고가가 지정가 이상이면 지정가 체결
- 같은 날 지정가와 별LOC가 모두 충족되면
  1. 지정가 75% 매도
  2. 남은 25% 별LOC 매도
  3. 주기 완료

## 추가된 전략

드롭다운에서 아래 항목을 선택합니다.

라오어 무한매수4.0 원문엔진 / Step24E

## 주의

- 30분할 별% 공식은 원문 확인 전까지 추가하지 않았습니다.
- TQQQ/SOXL 외 티커 공식은 원문 확인 전까지 추가하지 않았습니다.
- 후반전 세부 사다리 수량표는 원문 확인 전까지 임의 구현하지 않았습니다.
