# Step24S RAOR Original Engine

Step24S는 Step24Q/R 매매 엔진을 그대로 유지하면서 UI를 전환형 구조로 정리한 버전입니다.

## 변경점
- 좌측 사이드바 폭 축소
- 좌측 메뉴 클릭 시 앵커 스크롤 대신 섹션 전환
- 섹션: 대시보드 / Optimizer / 검증·로그 / 차트
- Step24S 전용 캐시 폴더: `step24_cache/step24s/`
- 24T 실시간 job_id + polling 구조로 확장하기 쉬운 프론트 구조 준비

## 주의
- 매매 공식은 변경하지 않았습니다.
- 라오어 원문에 없는 매수/매도 규칙은 추가하지 않았습니다.
