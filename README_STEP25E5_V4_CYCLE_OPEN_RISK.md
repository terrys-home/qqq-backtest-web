# Step25E-5 — RAOR V4.0 CycleStatus / OPEN 주기 / 장기주기 리스크 검증판

## 목적

Step25E-4에서 V4.0 일반모드의 완료주기 승률과 매도 이벤트 승률을 분리했다.
Step25E-5에서는 완료주기 승률을 유지하면서, 현재 미청산 OPEN 주기와 1000일 이상 장기주기를 별도 해석표로 분리한다.

## 변경 범위

- 매매 공식 변경 없음
- 체결 조건 변경 없음
- 지정가 매도율 원문 고정 유지
  - TQQQ: 15%
  - SOXL: 20%
- 리버스모드 제외 유지

## 추가 화면

### Step25E-5 V4.0 CycleStatus / OPEN 주기 / 장기주기 리스크 검증

추가 지표:

- COMPLETED 주기 수
- OPEN 주기 수
- REVIEW 주기 수
- 1000일 이상 장기주기 수
- 현재 OPEN 주기 번호
- OPEN 시작일 / 경과일
- OPEN T
- OPEN 평가액
- OPEN 수량

추가 표:

- 1000일 이상 장기 주기 상세
- 현재 OPEN 주기 상세
- 전체 주기 최종상태 요약

## 해석 기준

- COMPLETED: CycleCompleted 또는 IsCycleCompleted 이벤트가 확인된 주기
- OPEN: 백테스트 마지막 날 보유수량이 남아 있는 현재 주기
- REVIEW: 완료/진행 상태를 확정할 수 없는 주기

## 확인 포인트

1. 완료주기수와 매도 이벤트 수가 분리되어 있는지
2. 현재 OPEN 주기가 승률 계산에서 제외되어 있는지
3. REVIEW가 0인지
4. 1000일 이상 장기주기가 어떤 주기인지 확인 가능한지
5. OPEN 주기의 평가액과 T가 화면에 별도 표시되는지
