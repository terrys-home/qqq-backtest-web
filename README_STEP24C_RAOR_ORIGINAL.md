# Infinite Lab Step24C - 라오어 무한매수4.0 원문엔진 확장

## 기준
- Step24A/Step24B 파일은 유지하고 Step24C 파일을 추가했습니다.
- 원문에 없는 공식은 추가하지 않았습니다.
- 30분할 별% 공식, TQQQ/SOXL 외 티커 공식은 계속 차단합니다.

## Step24C 수정 내용
1. 드롭다운 정리
   - 현재 사용: RSI, Step24C, Step24B, Step24A
   - 이전 연구용: V1~V5, 오리지널 계열
   - 준비중: 떨사오팔, 종사종팔

2. 기간 설정 반영 방식 수정
   - 기존: 전체 데이터로 엔진 실행 후 화면에서 결과만 기간 필터
   - Step24C: 선택한 시작일~종료일 데이터를 먼저 잘라서 엔진 실행
   - 따라서 선택기간 이전 포지션/매수일이 매매로그에 섞이는 문제를 줄였습니다.

3. 매매로그 매수일 보정
   - 부분매도 로그의 `BuyDate`는 마지막 실제 매수체결일로 표시합니다.
   - 무한매수 한 주기의 최초 시작일은 `CycleStartDate`로 별도 기록합니다.
   - 기존처럼 모든 부분매도가 최초 진입일 하나로만 보이는 혼동을 줄였습니다.

4. 무한매수 완료 주기 분석 추가
   - `CycleId`
   - `CycleStartDate`
   - `CycleEndDate`
   - `CycleCompleted`
   - 기간 내 완료 주기 수
   - 완료 주기별 매수원금, 매도금액, 손익, 수익률, 주기일수

## Step24C 화면 보강 패치

- 상단 핵심 카드에 `완료 주기`, `진행 회차`, `진행 분할`을 추가했습니다.
- 미청산이 있는 경우 현재 진행 중인 무한매수 주기의 `현재 T / 분할횟수`, 진행률, 남은 분할, 진행 보유일을 표시합니다.
- 완료 주기는 `CycleCompleted=True`가 찍힌 `CycleId` 기준으로 중복 없이 집계합니다.
- 진행 회차는 마지막 백테스트 행의 `CycleId` 기준으로 표시합니다.

## Step24C UI 보강판

- 날짜 선택 UI를 브라우저 기본 date picker에서 연/월/일 직접 선택 방식으로 변경했습니다.
  - 연도 변경 시 긴 스크롤을 감을 필요 없이 드롭다운에서 바로 선택합니다.
  - 내부 전송값은 기존과 동일하게 `YYYY-MM-DD` 형식의 `start_date`, `end_date`입니다.
- 전략 선택 드롭다운에 `onchange` 자동 제출을 추가했습니다.
  - 무한매수 Step24C를 선택하면 분할횟수/첫매수 LOC 여유율 입력칸이 즉시 다시 렌더링됩니다.
  - 티커를 바꿨다가 돌아오거나 다른 전략을 눌렀다가 돌아오는 우회 동작이 필요 없도록 수정했습니다.
