# Step24W-2 Robust Mining 표시 보정판

수정 내용:

- `과최적화 방어 딥마이닝` 버튼 클릭 시 결과가 Optimizer 탭에 숨겨져 보일 수 있던 문제 보정
- `action_mode=robust` / `optimizer` / `deepmine` 실행 후 서버 렌더 단계부터 Optimizer 패널을 기본 활성화
- Robust 버튼 클릭 시에도 진행형 오버레이 표시
- 좌측 사이드바 문구를 Step24W 기준으로 정리
- 매매 엔진 공식 변경 없음
- Step24W Robust Mining 점수/후보표 로직 유지

확인 방법:

1. 전략을 `라오어 무한매수4.0 원문엔진 / Step24W`로 선택
2. `과최적화 방어 딥마이닝` 클릭
3. 계산 완료 후 자동으로 Optimizer 화면에서 `Step24W 과최적화 방어 딥마이닝` 카드와 `Robust Mining 후보표` 확인
